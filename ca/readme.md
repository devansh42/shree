# CA for shree
## Usage :
  * -addr string
    	(required) CA addr
  * -hprv string
    	(required) CA host private key
  * -hpub string
    	(required) CA host public key
  * -logdir string
    	CA log directory (default "App Dir")
  * -uprv string
    	(required) CA user private key
  * -upub string
    	(required) CA user public key
